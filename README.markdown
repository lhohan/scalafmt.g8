
A scalafmt giter8 template
----------

- `sbt -sbt-version 0.13.15 new lhohan/scalafmt.g8`

Note:
- Auto-formatting on compile is enabled.
- Default formatting style is 'default-with-align'.

Also configured:
- coursier: Faster artifact fetching
